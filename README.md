# weatherstation

## Installation

```bash
$ sudo apt install python3-venv
$ python3 -m venv venv
$ . venv/bin/activate
$ pip3 install flask
```
## Start
```bash
$ export FLASK_APP=weatherstation
$ flask run
```



## Production

```bash
$ sudo mkdir /opt/weatherstation
$ sudo apt install python3-venv
$ cd /opt/weatherstation
```
